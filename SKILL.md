# 🦞 A股量化监控系统

## 快速开始

### 查询当前行情
```bash
uv run {baseDir}/core.py
```

### 修改监控配置
编辑 `{baseDir}/config.ini`：
- 股票代码和名称
- 涨跌提醒阈值
- 目标价格
- 技术指标参数

## 功能

### 1. 实时行情
- 当前价、开盘价、最高价、最低价
- 使用15分钟线，接近实时

### 2. 价格提醒
| 告警类型 | 条件 |
|---------|------|
| 涨跌提醒 | 涨跌幅超过阈值 |
| 低于目标价 | 价格 < target_low |
| 高于目标价 | 价格 > target_high |

### 3. 技术指标信号

| 指标 | 买入信号 | 卖出信号 |
|------|---------|---------|
| **MA均线** | 快线金叉慢线 | 快线死叉慢线 |
| **MACD** | DIF上穿DEA | DIF下穿DEA |
| **RSI** | < 30 超卖 | > 70 超买 |
| **KDJ** | K、D<20超卖 或 金叉 | K、D>80超买 或 死叉 |
| **布林带** | 触及下轨 | 触及上轨 |

### 4. 交易建议
- 操作方向：买入/卖出/持有
- 建议价格
- 止损/止盈位
- 仓位建议（基于信号强度）

## 配置文件

`config.ini` 关键配置：

```ini
[stock]
code = sh600021      # 股票代码
name = 上海电力       # 股票名称

[alert]
change_threshold = 1.0   # 涨跌阈值%
target_low = 19.0        # 低于此价提醒
target_high =            # 高于此价提醒（留空禁用）

[indicators]
ma_fast = 5        # 快速均线
ma_slow = 10       # 慢速均线
rsi_oversold = 30  # RSI超卖阈值
rsi_overbought = 70 # RSI超买阈值
```

## 信号权重

| 信号数量 | 买入仓位 | 卖出减仓 |
|---------|---------|---------|
| 3+个信号 | 50-70% | 50-70% |
| 2个信号 | 30-50% | 30-50% |
| 1个信号 | 10-20% | 10-20% |

## 定时监控

系统已配置 crontab，交易时间每5分钟自动检查。

查看配置：
```bash
crontab -l
```

## 注意事项

- 数据源：新浪/腾讯双核心，自动故障切换
- 分钟数据有约15分钟延迟
- 止损/止盈仅供参考，请结合自身风险承受能力调整
- 技术指标不构成投资建议，仅供研究参考
